## 작업 내용

### Phase 0 완료 — 인증 · 온보딩 플로우

**Expo SDK 다운그레이드 (56 → 54)**
- Expo Go(폰)가 SDK 54를 지원하는데 프로젝트가 SDK 56이어서 호환 오류 발생
- `npx expo install expo@^54.0.0` + `--fix`로 전체 의존성 정렬
- peer dependency 충돌은 `--legacy-peer-deps`로 해결

**DB 트리거 버그 수정**
- 신규 가입 시 `AuthApiError: Database error saving new user` 발생
- 원인: `token_balances.user_id`가 `auth.users`가 아닌 `profiles.user_id`를 FK 참조하는데, 트리거가 `profiles` 행 없이 `token_balances`에 먼저 INSERT 시도
- 1차 수정: `handle_new_user()` 트리거에 `profiles` 빈 행 INSERT 추가
- 2차 수정: `profiles.gender`, `birth_year`를 nullable로 변경 (온보딩에서 채우는 구조)

**로그인 후 화면 미전환 버그 수정**
- `sign-in.tsx`가 성공 후 아무 네비게이션도 호출하지 않음
- `onAuthStateChange` 의존 구조였으나 `index.tsx`가 마운트되지 않은 상태라 무반응
- `router.replace('/')` 명시적 호출로 해결

**비밀번호 입력창 UX 개선**
- `secureTextEntry` 필드에 자동 대문자 방지 (`autoCapitalize="none"`)
- 보기/숨기기 토글 버튼 추가 (`TextField` 컴포넌트 공통 처리)
